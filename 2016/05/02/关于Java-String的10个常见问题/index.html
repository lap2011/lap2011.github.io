1、怎么对比两个String，用”==”或者用equals()

总之，如果是想对比是否是同一个对象用”==”，如果相对比值是否相等用equals()。除非你是想要对比两个字符串对象是否相等，一般都应该使用equals()。最好你能了解原理。

2、为什么char数组没有string适合存储安全信息

字符串是不可变的，意味着一旦String对象被创建，在被垃圾回收前都不会再改变。但是如果使用的是char数组，就可以随意改变其中的元素。如果通过这种方式，安全的敏感信息(比如密码)就不能在系统中任意一个地方存储了。

3、String是否可用于switch呢

在jdk1.7之后就可以使用string作为switch的条件，在1.6之前是不可以的

6、subString()方法实际在做什么事

在jdk1.6中，subString()通过在现有字符串的char数组中移动一个窗口来截取，而不是创建一个新的对象。创建一个新的字符串的时候，新的字符串数组的value仍然指向了原来的char数组的内存。上面的方法可能是代码运行很快，但是可能会导致垃圾回收的时候保留了很大的字符串。

在jdk1.7中，substring()创建一个新的char数组，不会使用当前的对象，可以通过这篇文章对比6和7的区别

7、String vs StringBuilder vs StringBuffer

String vs StringBuilder: StringBuilder是可变的，这意味着你在创建它之后可以修改它

StringBuilder vs StringBuffer:StringBuffer是同步的，是线程安全的，但是效率低于StringBuilder

8、怎么重复一个字符串

在python中，只需要乘以一个数字来重复一个字符串。在java中，我们可以调用StringUtils的repeat()方法来重复一个字符串。
