Python装饰器基本理解

发表于 2014-06-14 | 浏览 次

﻿﻿﻿﻿装饰器的亮点作用：

插入日志、性能测试、事务处理

装饰器的作用:为已经存在的对象添加额外的功能

类似与c++的回调函数

修饰符等价于包装调用

修饰函数必须返回一个“可调用对象”

内置的装饰器有三个，分别是staticmethod、classmethod和property，作用分别是把类中定义的实例方法变成静态方法、类方法和类属性

当有多个装饰器的时候，会从下往上调用，也就是调用离函数最近的一个装饰器

执行流程是：
​ 1、调用print function
​ 2、调用timeslong
​ 3、调用timeslong.call
​ 4、调用function
​ 5、调用call的剩余部分
​ 6、最后timeslong返回了call的返回结果，然后print了出来
